Кроссплатформенная среда выполнения для JavaScript, которая работает на серверах.

React

JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов.

Bootstrap

Фреймворк интерфейсных компонентов для создания интерактивных и адаптивных веб-приложений и веб-сайтов.

Gulp

Таск-менеджер для автоматического выполнения часто используемых задач, написанный на языке программирования JavaScript.

Инструмент веб-разработчика, который соединяет JavaScript- и CSS-файлы в единое целое — файл.

FIgma
